# Professional Interactive To-Do List with Drag-and-Drop Functionality

A fully interactive To-Do List web application built with **React** and **TypeScript**, styled using **Tailwind CSS**. This app allows users to create, display, reorder, and manage tasks dynamically with **drag-and-drop functionality** in a clean and responsive user interface.

## Project Objective
This project demonstrates the implementation of:

- React state management
- DOM manipulation
- User experience (UX) best practices
- Smooth interactive drag-and-drop features

## Features

### Add Tasks
- Users can add new tasks using an input field and **Add** button.
- Blank inputs are not accepted to maintain task integrity.

### Display Tasks
- All tasks are displayed in a vertical list with clear separation.
- Each task includes its name and a **delete button** for easy removal.

### Delete Tasks
- Users can remove any task from the list with a single click.

### Reorder Tasks (Drag & Drop)
- Tasks can be reordered via drag-and-drop.
- Dropped tasks swap positions cleanly without duplicates or blank spaces.
- List order updates instantly after drop.

### UI/UX
- Clean, modern, and professional color palette.
- Smooth hover and drag animations for better interactivity.
- Responsive layout that works well on desktop and mobile devices.

## Recommended Tech Stack
- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS
- **Optional:** LocalStorage for task persistence

## Bonus Features (Optional)
- Save tasks to localStorage to persist on page reload.
- Add task completion with checkbox and strike-through.
- Organize tasks into categories like "Today" or "Tomorrow".
